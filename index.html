Application 6: Theme Park Ride Control System

A small demo site for the project — contains a short embedded video demo, Wokwi link, and concise project description.

Project description

This project is a real-time system that represents a prototype roller coaster ride control system inspired by Disney attractions. The system consists of three main components: an ultrasonic distance sensor, an emergency stop push button, and multiple LEDs. These components are programmed on an ESP32 microcontroller to work together to ensure a smooth and safe ride experience.

The ultrasonic distance sensor detects if any obstacles are near the ride vehicle and, when triggered, alerts the operator by flashing a blue LED. The emergency stop push button immediately halts the ride for three seconds and activates the red LED to indicate a safety stop. Meanwhile, the green LED signals that the ride is operating safely and functioning normally.

By using FreeRTOS, the system ensures that all critical tasks meet their deadlines and operate reliably, which is essential for real-time safety-critical applications like ride control systems.

Quick highlights
  • Simulated ride control using an embedded microcontroller (Wokwi simulation).
  • Safety features: emergency stop, interlock checks, and status LEDs.
  • Demo video shows end-to-end operation including faults and recovery.
